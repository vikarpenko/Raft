# 🦦 Raft
Raft is a web application for planning and organizing students' personal and academic life.

## Tech Stack

### Frontend
- React
- TypeScript
- Vite

### Backend
- Java 21
- Spring Boot
- Maven

## Getting Started

### Backend
```bash
cd backend
./mvnw spring-boot:run
```

### Docker
```bash
cd backend
docker-compose up -d
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
